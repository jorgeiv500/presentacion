Objetivos

  • Algoritmos supervisados y no supervisados
  • Preprocesamiento y selección de características
  • Clasificación, regresión y clustering

Importancia del preprocesamiento

Estandarizar, normalizar y limpiar los datos antes de aplicar los modelos.

Selección de características relevantes

Eliminar ruido y redundancia para mejorar rendimiento.

Algoritmos supervisados y no supervisados

Supervisados utilizan datos etiquetados, no supervisados encuentran patrones.

Importar datos


					import pandas as pd
					from sklearn.model_selection import train_test_split
					
					df = pd.read_csv("data.csv")
					X = df[["var1", "var2"]]
					y = df["target"]
					
					X_train, X_test, y_train, y_test = train_test_split(X, y)

Preprocesamiento - Estandarización


					from sklearn.preprocessing import StandardScaler
					
					scaler = StandardScaler()
					X_train = scaler.fit_transform(X_train)
					X_test = scaler.transform(X_test)

Selección de características


					from sklearn.feature_selection import SelectKBest
					from sklearn.feature_selection import f_classif
					
					selector = SelectKBest(f_classif, k=3)
					selector.fit(X, y)

Clasificación


					from sklearn.ensemble import RandomForestClassifier
					
					model = RandomForestClassifier()
					model.fit(X_train, y_train)

Regresión


					from sklearn.linear_model import LinearRegression
					
					linreg = LinearRegression()
					linreg.fit(X_train, y_train)
