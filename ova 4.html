Estadística Inferencial con Python

Objetivos

  • Aplicar pruebas de hipótesis
  • Estimar intervalos de confianza
  • Analizar varianza con ANOVA

Prueba T de Student


					from scipy import stats
					
					t_stat, p_val = stats.ttest_ind(a, b)
					if p_val < 0.05:
						print("Diferencia significativa")

Compara medias de dos muestras independientes.

Supuestos prueba T

  • Muestras independientes
  • Datos continuos
  • Distribución normal
  • Varianzas iguales

Prueba Chi Cuadrado


					from scipy import stats
					
					chi2, p_val, df, expected = stats.chi2_contingency(data) 
					if p_val < 0.05:
						print("Relación significativa")

Analiza relación entre variables categóricas.

Supuestos Chi Cuadrado

  • Observaciones independientes
  • Categorías mutuamente excluyentes
  • Frecuencias esperadas mayores a 5

Intervalos de confianza


					import scipy.stats as st
					
					ic = st.norm.interval(0.95, loc=mean, scale=std)
					print(f"IC 95%: {ic}")

Estima rango de valores de un parámetro poblacional.

Supuestos Intervalos de Confianza

  • Muestra aleatoria
  • Tamaño muestral suficiente
  • Datos continuos

ANOVA

 
					from scipy import stats
					
					f_stat, p_val = stats.f_oneway(a, b, c)
					if p_val < 0.05:
						print("Diferencia significativa entre medias")

Compara medias de 3 o más grupos.

Supuestos ANOVA

  • Distribución normal
  • Varianzas iguales
  • Observaciones independientes

Correlación de Pearson


					from scipy import stats
					
					corr, p_val = stats.pearsonr(x, y)
					if p_val < 0.05:
						print("Correlación significativa")

Mide relación lineal entre dos variables cuantitativas.

Correlación de Spearman

  
					from scipy import stats
					
					rho, p_val = stats.spearmanr(x, y)
					if p_val < 0.05:
						print("Correlación significativa")

Mide relación monótona entre dos variables cuantitativas.
